# 완료한 작업
GitHub Action 프로세스에 대한 조사 및 문서 작성

브랜치에 push 이벤트가 발생한 이후 개발서버 배포까지 프로세스 설명
1. 어떻게 push 이벤트를 감지?
2. 이벤트를 감지한 이후의 동작은 어디에 어떻게 작성?
3. 환경변수 설정은 언제 어디서 이루어지나?
4. 빌드한 파일은 어디로 언제 업로드 되며, 언제 다운로드 받아지나?
5. 다운로드 받은 artifact(산출물)은 어떻게 확인할 수 있나?
